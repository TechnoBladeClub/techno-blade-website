
# Techno Blade Club Website

> Official website for **Techno Blade Club** – to introduce the club and showcase its activities and news.

---

## 🎯 Project Goal

- Build a modern landing page for the club
- Showcase club activities and workshops
- Provide ways to join and contact the club

---

## 📌 Project Roadmap

### Phase 1: Landing Page
- Hero Section
- About Section
- Activities Section
- Join Us Section
- Navbar & Footer
- Responsive Design
- Deployment (Vercel)

### Phase 2: Blog / Articles (Later)
- Display educational articles / club news
- Categorize articles
- Content management

---

## 🛠 Tech Stack

- **Frontend:** Next.js + Tailwind CSS
- **Backend:** API Routes within Next.js (if needed)
- **Deployment:** Vercel
- **Version Control:** Git + GitHub Flow

---

## 📂 Suggested Project Structure

```

/app
/components

/lib
/styles

````

---

## 🏷 GitHub Flow Workflow

1. Each feature in a separate branch:
   ```bash
   git checkout -b feature/<feature-name>
````

2. Link each commit to its Issue:

   ```
   git commit -m "feat: add navbar #12"
   ```
3. Open a Pull Request
4. Review → Merge into main

---

## 📋 Contribution Guidelines

* Each member picks an Issue to work on
* No merges without a Pull Request
* Follow the branch naming convention
